void SharedBuffer::buildFromPublishers ( const std::vector< std::unique_ptr< SvPublisherInstance > > & publishers)

Build the interleaved schedule from all publishers.

Merges frames from every READY publisher, sorted by timestamp. Publishers are staggered so their packets interleave evenly.
