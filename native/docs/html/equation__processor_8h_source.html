2 * @file equation_processor.h
3 * @brief Equation Processing for SV Publisher
4 *
5 * Parses waveform equations and generates samples for IEC 61850-9-2LE.
6 *
7 * Supported equation format:
8 * "100 * sin(2 * PI * 50 * t)" -> amp=100, freq=50 Hz
9 * "325 * sin(2 * PI * 50 * t - 2*PI/3)" -> amp=325, freq=50 Hz, phase=-120°
