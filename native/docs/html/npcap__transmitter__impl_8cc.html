Detailed Description

Npcap Network Transmitter.

This module provides raw Ethernet packet transmission using the Npcap library. Supports both single-packet and high-speed batch transmission modes.

Features:

  • Dynamic DLL loading (Windows)
  • Interface enumeration with MAC address detection
  • Single packet transmission
  • SendQueue batch transmission for high throughput
