Overview

Building

The native module is built as part of the Tauri build process:

cd src-tauri
cargo build --release

Dependencies

  • Npcap SDK - Raw packet capture/transmission
  • C++17 - Modern C++ features
  • Windows.h - Windows API (timers, threading)
