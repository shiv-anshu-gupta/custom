Detailed Description

Equation Processing for SV Publisher.

Parses waveform equations and generates samples for IEC 61850-9-2LE.

Supported equation format: "100 * sin(2 * PI * 50 * t)" -> amp=100, freq=50 Hz "325 * sin(2 * PI * 50 * t - 2*PI/3)" -> amp=325, freq=50 Hz, phase=-120° "0" -> zero signal
