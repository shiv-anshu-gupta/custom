1/**
2 * @file sv_publisher_instance.h
3 * @brief Single SV Publisher Instance (one Merging Unit)
4 *
5 * Each SvPublisherInstance represents one simulated Merging Unit with:
6 * - Its own configuration (svID, appID, MAC, channels, sample rate)
7 * - Its own EquationProcessor (generates waveform samples)
8 * - Its own internal buffer (pre-built frames for one AC cycle)
9 *
10 * Architecture:
11 * SvController creates multiple SvPublisherInstance objects.
12 * Each instance pre-builds its frames. The controller then merges
13 * all frames into a SharedBuffer for transmission.
14 *
15 * ┌─────────────────────┐
16 * │ SvPublisherInstance │
17 * │ ┌───────────────┐ │
18 * │ │EquationProc. │──┼──→ Internal Buffer (pre-built frames)
19 * │ │(own instance) │ │ │
20 * │ └───────────────┘ │ ▼
21 * │ Config: svID, etc. │ SharedBuffer (merged schedule)
22 * └─────────────────────┘ │
23 * ▼
24 * npcap writer
25 */
